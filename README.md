# Gamer Store
<b> <h1>#  Projeto 4 - Gamer Sotre </h1> </b>
 
 <hr size=8 width=50% align=right>

 <p align="justify"> Este repositório foi criado para o desenvolvimento de um algoritmo para o cliente GAMER STORE. O algorítimo deve ser capaz de calcular o valor a ser pago por um produto considerando o seu preço normal e condições de pagamento. [ 1 ] A vista dinheiro/ cheque: 10% de desconto, [ 2 ] A vista no cartão: 5% de desconto, [ 3 ] Em até duas vezes no cartão: preço normal, [ 4 ] 3X ou mais no cartão: 20% de juros." </p>
